Google Fonts Flicker

Please refresh the page a few times

Experience may vary by web browser

Please refresh this page several times to make sure you can see the flickering issue. In some cases, you may need to hold down the Shift key while clicking the Refresh button. Generally, you should see a sans-serif font for a second and then it is replaced by the Google-provided font Source Sans Pro.

How would you fix this?

Please view and copy the source, find a solution, and post either a link to your solution or the entire source code of your solution into your application.
